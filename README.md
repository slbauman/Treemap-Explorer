# Treemap Explorer
Working name for project.

This program will visualize the size of files and folders on a filesystem. It
is currently a work in progress.

![Screenshot](screenshot.png?raw=true "Treemap Explorer Screenshot")

## Dependencies:
* SDL2
* SDL2_ttf

## Usage:
`make && ./tme`
